### Review Notes
**Copy and paste this into App Store Connect:**

```
BOJANG - TIBETAN LANGUAGE LEARNING APP

APP OVERVIEW:
Bojang is an educational app designed to teach the Tibetan language to English speakers through interactive quizzes. The app features a progressive learning system with three skill levels and 26 topic-based lessons.

KEY FEATURES TO TEST:
1. Launch the app - no login required
2. Navigate through the level selection screen
3. Try quizzes from different levels (Beginner, Intermediate, Advanced)
4. Test audio feedback by answering questions correctly and incorrectly
5. Check progress tracking and scoring system
6. Test navigation between different quiz topics

CONTENT INFORMATION:
- All content is educational and culturally appropriate
- Tibetan script and English translations are provided
- Audio files contain pronunciation feedback
- No user-generated content
- No social features or sharing capabilities
- No in-app purchases or subscriptions

TECHNICAL NOTES:
- App works offline after initial installation
- No network connectivity required for core functionality
- No data collection or user tracking
- Local storage only for progress tracking
- Compatible with iOS devices

CULTURAL SENSITIVITY:
- Content has been developed with respect for Tibetan culture
- Focuses on language learning and cultural appreciation
- Includes traditional greetings and cultural contexts
- Suitable for all ages and backgrounds

NO ISSUES EXPECTED:
- No controversial content
- No violence, adult content, or inappropriate material
- No gambling or simulated gambling
- No medical or health claims
- No political content
- Complies with all App Store guidelines

TESTING SUGGESTIONS:
1. Start with Level 1 (Beginner) → Alphabet quiz
2. Try Level 2 (Intermediate) → Animals or Emotions quiz  
3. Test Level 3 (Advanced) → Introducing or Order Food quiz
4. Verify audio plays for correct/incorrect answers
5. Check that progress is saved between sessions
6. Test app behavior when returning from background

The app is ready for immediate approval and distribution.
```

3. Test thoroughly before resubmission
4. Update review notes if necessary
5. Resubmit with clear explanation of changes made
