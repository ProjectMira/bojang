# App Store Screenshots Guide for Bojang
## Complete Requirements and Best Practices

### Screenshot Requirements Overview

**Required Sizes for iPhone:**
- **6.7" Display** (iPhone 14 Pro Max, 13 Pro Max, 12 Pro Max): 1290 x 2796 pixels
- **6.5" Display** (iPhone 14 Plus, 13, 12, 11): 1242 x 2688 pixels  
- **5.5" Display** (iPhone 8 Plus, 7 Plus, 6s Plus): 1242 x 2208 pixels

**Required Sizes for iPad (if supporting iPad):**
- **12.9" Display** (iPad Pro 12.9"): 2048 x 2732 pixels
- **11" Display** (iPad Pro 11", iPad Air): 1668 x 2388 pixels

**File Format:** PNG or JPEG
**Color Space:** RGB
**Maximum File Size:** 8MB per screenshot
**Minimum Required:** 3 screenshots
**Maximum Allowed:** 10 screenshots

### Recommended Screenshot Sequence

**Screenshot 1: App Launch/Home Screen**
- **Content:** Level selection screen showing all three levels
- **Purpose:** First impression, shows app structure
- **Text Overlay:** "Learn Tibetan with Interactive Quizzes"
- **Key Elements:** Clean interface, level progression, app logo

**Screenshot 2: Quiz Interface**
- **Content:** Active quiz screen with Tibetan text and multiple choice options
- **Purpose:** Shows core functionality
- **Text Overlay:** "Master Tibetan Script & Vocabulary"
- **Key Elements:** Tibetan script, answer options, progress bar

**Screenshot 3: Audio Feedback**
- **Content:** Feedback screen showing correct answer celebration
- **Purpose:** Highlights audio learning feature
- **Text Overlay:** "Audio Feedback for Perfect Pronunciation"
- **Key Elements:** Success animation, Tibetan congratulations text

**Screenshot 4: Progress Tracking**
- **Content:** Score display or completion screen
- **Purpose:** Shows learning progress features
- **Text Overlay:** "Track Your Learning Journey"
- **Key Elements:** Score display, progress indicators

**Screenshot 5: Content Overview**
- **Content:** Sublevel selection showing topic variety
- **Purpose:** Demonstrates content breadth
- **Text Overlay:** "26 Topics from Alphabet to Conversations"
- **Key Elements:** Topic list, level organization

### Screenshot Content Guidelines

**What to Include:**
✅ **Tibetan Script Prominently:** Show authentic Tibetan text
✅ **Clean Interface:** Highlight intuitive design
✅ **Cultural Elements:** Include respectful cultural content
✅ **Progress Indicators:** Show learning advancement
✅ **Audio Visual Cues:** Indicate sound/pronunciation features
✅ **Multiple Levels:** Demonstrate progression system
✅ **Engaging Content:** Show variety of topics

**What to Avoid:**
❌ **Empty States:** No blank or loading screens
❌ **Error Messages:** No crashes or bugs visible
❌ **Personal Information:** No test data or personal details
❌ **Placeholder Content:** All content should be final
❌ **Low Quality Images:** Ensure crisp, clear screenshots
❌ **Inconsistent UI:** Use same app version for all screenshots
❌ **Offensive Content:** Nothing inappropriate or controversial

### Text Overlay Best Practices

**Design Guidelines:**
- Use readable fonts (San Francisco, Helvetica)
- High contrast colors (white text on dark overlay)
- Keep text concise (5-7 words maximum)
- Position text in safe areas (avoid notch/home indicator)
- Consistent styling across all screenshots

**Recommended Text Overlays:**

1. **"Learn Tibetan Through Interactive Quizzes"**
2. **"Master Authentic Tibetan Script"**
3. **"Audio Feedback for Perfect Pronunciation"**
4. **"Track Your Progress & Celebrate Success"**
5. **"26 Topics from Basics to Conversations"**

### Screenshot Creation Process

**Step 1: Prepare Your Device**
- Use iPhone with highest resolution available
- Ensure app is installed and fully functional
- Clear notifications and set clean status bar
- Use consistent time (9:41 AM is Apple's standard)
- Full battery indicator
- Strong signal strength

**Step 2: Capture Screenshots**
- Take screenshots in portrait orientation
- Capture at native device resolution
- Ensure UI elements are fully visible
- Take multiple versions of each screen
- Include different quiz topics/levels

**Step 3: Edit and Enhance**
- Add text overlays using design tools
- Ensure consistent branding
- Optimize for App Store display
- Check readability on different backgrounds
- Verify file sizes are under 8MB
